# Simple Poll App

## Project Description
This project is a simple poll application where users can create polls, vote on them and see results.

## Features
- User accounts
- Create, edit and close polls
- Voting
- Share polls using a link
- Offline support (PWA)

## Tech Stack
- Client: JavaScript
- Server: Node.js with Express
- Database: PostgreSQL
- Version control: Git and GitHub
