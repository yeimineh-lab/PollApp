Welcome to Simple Poll

Simple Poll lets you create polls quickly, browse public polls, and vote anonymously.

How the app works

You can use Simple Poll as a guest or with an account. Guests can create public polls and vote on public polls. If you create an account, you can also manage your profile and access account-only polls.

Public polls

Public polls are visible to everyone. Anyone can open them and vote anonymously. To see them, go to the Public Polls page.

Creating polls

You can create a guest poll without logging in. If you want more control and access to account-based features, create an account and sign in.

Voting

Voting is anonymous. The app shows totals and percentages, not who voted. Public guest voting is limited in the browser so the same device cannot easily vote multiple times on the same poll.
